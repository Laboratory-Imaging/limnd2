# limnd2 examples

This folder contains example scripts demonstrating various features of the `limnd2` library for working with ND2 files.

## Example scripts

| Script                                                                 | Description                                                                                  |
|------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [`convert_to_nd2_examples`](./convert_to_nd2_examples)                 | Collection of scripts showing how to convert different image formats into ND2 files.         |
| [`deepsim_reader_example.py`](./deepsim_reader_example.py)             | Demonstrates reading ND2 files with DeepSIM data and how to access them                      |
| [`example_reader.py`](./example_reader.py)                             | Basic example of reading ND2 files and accessing their data and metadata.                    |
| [`example_writer.py`](./example_writer.py)                             | Shows how to write image data and metadata into a new ND2 file.                              |
| [`experiment_factory_example.py`](./experiment_factory_example.py)     | Illustrates creating ND2 experiment structures programmatically.                             |
| [`metadata_factory_example.py`](./metadata_factory_example.py)         | Demonstrates building and attaching custom metadata to ND2 files.                            |

Each script serves as a practical example of using `limnd2` for different ND2 file operations. See the individual scripts for usage details and specific implementation.
